## Getting Started in 3 Steps

### Step 1: Choose Your Template
1. Open the builder at `/builder`
2. Click **"Templates"** button in the header
3. Select a template that matches your profile:
   - **Developer** - For software engineers
   - **Designer** - For UI/UX designers  
   - **Student** - For students/graduates
   - **Founder** - For entrepreneurs
4. Or choose "Start from Scratch"

### Step 2: Customize Your Content
1. Click any block on the canvas to edit it
2. Update your information in the right sidebar:
   - **Header**: Your name, bio, location, avatar
   - **Skills**: Add your skills with proficiency levels
   - **Experience**: Your work history or education
   - **Projects**: Showcase your best work
   - **Links**: Social media and portfolio links

### Step 3: Publish & Share
1. Click **"Publish"** in the header
2. Your profile is instantly live at `/u/your-handle`
3. Share your portfolio URL!

## ðŸŽ¨ Customization Guide

### Skill Proficiency Levels

**When to use each level:**
- **Beginner** ðŸŸ¡ - Learning, basic understanding, <1 year
- **Intermediate** ðŸ”µ - Comfortable, practical experience, 1-3 years
- **Advanced** ðŸŸ¢ - Expert, production experience, 3+ years

### Experience Formatting

**Date formats that work well:**
- "Jan 2023" or "January 2023"
- "2023" for year only
- Leave end date empty for current positions
- Use "Present" if you prefer explicit indication

### Project Best Practices

**Make projects stand out:**
- Choose a clear, descriptive title
- Write 1-2 sentence descriptions (not paragraphs)
- Add 3-5 relevant tech stack items
- Include project images when possible (16:9 ratio works best)
- Link to live demos or GitHub repos

## ðŸ”§ Advanced Features

### Keyboard Shortcuts
- **Click** a block from library to add it
- **Drag** blocks to reorder them on canvas
- **Undo/Redo** buttons in header for mistakes
- **Delete** via inspector panel (right sidebar)

### Editing Complex Blocks

**Skills Block:**
- Click "Add Skill" to create new entries
- Use the dropdown to set proficiency level
- Drag the grip handle to reorder (coming soon)
- Delete with trash icon

**Experience Block:**
- Fill in all fields for each position
- Description supports multi-line text
- Leave end date empty for current roles
- Add multiple positions easily

**Projects Block:**
- Type tech names and press Enter to add them
- Click X on tech tags to remove them
- Images are optional but recommended
- Links open in new tab

## ðŸ†˜ Troubleshooting

**Profile not saving?**

- LocalStorage must be enabled
- Clear cache and reload if needed

- Check that you clicked/dragged correctly
